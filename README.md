# Game-Storage-App
Aplicação web desenvolvida em Java com Spring Boot e Thymeleaf para cadastrar e visualizar jogos. O sistema permite armazenar informações básicas sobre cada jogo e exibi-las em formato JSON.

Funcionalidades

  Cadastrar novos jogos com:

- Título

- Editora

- Gênero

- Ano de lançamento

- -  Visualizar lista completa dos jogos cadastrados

- - Exibir todos os jogos em formato JSON

- - Navegação simples entre as páginas

  Tecnologias usadas:

- Java

- Spring Boot

- Thymeleaf

- HTML

- Maven

  Estrutura do projeto

- index.html — Página inicial com links para todas as funções

- create.html — Formulário para cadastro de novos jogos

- list.html — Exibe a lista dos jogos registrados

- GameViewController.java — Controla as rotas e páginas de visualização

- GameJSONController.java — Controla o retorno dos dados em formato JSON
